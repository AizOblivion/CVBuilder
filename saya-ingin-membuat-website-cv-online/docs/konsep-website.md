# Konsep Website Pembuatan CV Online

## 1. Posisi Website

Website ini bukan sekadar landing page biasa, tetapi halaman pemasaran layanan yang membantu pengunjung cepat memahami:

- apa layanan yang ditawarkan
- manfaat yang akan didapat pengguna
- bagaimana proses pembuatan CV berjalan
- bagaimana cara mulai menggunakan layanan

## 2. Target Pengunjung

- fresh graduate
- job seeker
- profesional yang ingin update CV
- freelancer
- pencari kerja yang ingin CV lebih rapi dan ATS-friendly

## 3. Tujuan Utama

- menjelaskan value layanan dalam beberapa detik pertama
- meningkatkan kepercayaan pengunjung
- menunjukkan hasil dan kualitas template
- mengarahkan pengunjung ke aksi: mulai buat CV, pilih template, hubungi admin

## 4. Arah Visual

Tema visual:

- modern
- profesional
- tepercaya
- efisien

Karakter tampilannya:

- headline besar dan kuat
- layout luas dan fokus CTA
- warna netral dengan aksen hangat
- card untuk fitur, langkah, dan template

## 5. Rekomendasi Konten per Section

### Hero

Isi:

- headline manfaat utama
- subheadline singkat
- tombol "Mulai Buat CV" dan "Lihat Template"
- preview hasil CV atau mockup dashboard

Contoh arah copy:

> Buat CV online yang lebih profesional, rapi, dan siap kirim kerja dalam hitungan menit.

### Kenapa Perlu CV Online

Isi:

- masalah yang sering dialami pengguna
- alasan CV modern penting
- penjelasan nilai layanan

### Fitur

Isi:

- template profesional
- edit data dengan mudah
- export PDF
- desain ATS-friendly
- akses online

### Cara Kerja

Sajikan dalam 3 langkah:

1. isi data
2. pilih template
3. unduh atau bagikan CV

### Template / Hasil CV

Setiap kartu proyek sebaiknya memuat:

- nama template
- gaya visual
- target pengguna
- preview singkat

### Testimoni / FAQ

Bisa berisi:

- testimoni singkat pengguna
- pertanyaan umum
- jaminan revisi atau dukungan

### Kontak

Isi:

- email
- WhatsApp
- Instagram / LinkedIn brand
- form konsultasi singkat

## 6. Struktur Informasi

```text
Navbar
|- Home
|- Fitur
|- Cara Kerja
|- Template
|- FAQ
`- Contact

Main
|- Hero
|- Problem / Value
|- Metrics
|- Features
|- Steps
|- Templates
|- Testimonials / FAQ
`- Contact CTA

Footer
|- Copyright
|- Social Links
`- Quick Navigation
```

## 7. Aset yang Sebaiknya Disiapkan

- logo brand sederhana
- preview template CV
- ilustrasi / mockup CV
- daftar harga atau paket
- link WhatsApp dan media sosial brand

## 8. Tahap Pengerjaan

1. Menentukan positioning brand dan target pengguna
2. Menulis copy landing page
3. Menyusun layout desktop dan mobile
4. Menambahkan template, harga, dan CTA
5. Optimasi performa dan SEO dasar
6. Deploy
